## Fashion-MNIST 数据集 (书上默认)

### 1. 数据集简介
- 28x28 灰度图, 10 类
- 训练集 60,000 张, 测试集 10,000 张
- 比 MNIST 更难一点(因为衣服确实更像一坨)

类别顺序(书上固定):
```
0: T-shirt/top
1: Trouser
2: Pullover
3: Dress
4: Coat
5: Sandal
6: Shirt
7: Sneaker
8: Bag
9: Ankle boot
```

---

### 2. 读取数据 + DataLoader
```python
import torch
from torch.utils import data
import torchvision
from torchvision import transforms

trans = transforms.ToTensor()

train = torchvision.datasets.FashionMNIST(
    root = "../data",
    train = True,
    transform = trans,
    download = True,
)

test = torchvision.datasets.FashionMNIST(
    root = "../data",
    train = False,
    transform = trans,
    download = True,
)

batch_size = 256
train_iter = data.DataLoader(train, batch_size = batch_size, shuffle = True)  # Windows 可能要加 num_workers = 0
test_iter = data.DataLoader(test, batch_size = batch_size, shuffle = False)   # 同上
```

### 3. 看几张图(可视化)
```python
import matplotlib.pyplot as plt

labels = [
    "t-shirt", "trouser", "pullover", "dress", "coat",
    "sandal", "shirt", "sneaker", "bag", "ankle boot",
]

def get_fashion_mnist_labels(y):
    return [labels[int(i)] for i in y]

# 取一个小批量显示
X, y = next(iter(train_iter))

plt.figure(figsize = (8, 4))
for i in range(6):
    plt.subplot(2, 3, i + 1)
    plt.imshow(X[i].squeeze(), cmap = "gray")
    plt.title(get_fashion_mnist_labels(y)[i])
    plt.axis("off")
plt.show()
```
